# pagezero

> Build your startup with a tiny team, fast!

The open-source, full-stack TypeScript project starter focused on speed of development.

Guiding principles:

- Easy to learn
- Fast to build
- Pleasure to develop
- Cheap to maintain

Built on strong foundations:

- Vite
- React
- Cloudflare Infra
- TypeScript
- Github Actions
- TailwindCSS
- React Router v7
- Drizzle ORM
- Prettier
- ESLint
- Vitest
- Playwright
- Storybook
- MDX
