This is an example blog post for the PageZERO web app template. Posts live as `.mdx` files in `apps/blog/routes/posts/` — drop in a new file and it appears on the blog index automatically.

## What you get out of the box

- **MDX authoring** — write posts in Markdown with embedded JSX
- **Styled typography** — headings, paragraphs, lists, and tables map to the design system
- **Syntax highlighting** — fenced code blocks are highlighted via `rehype-highlight`
- **Frontmatter** — export metadata (title, date, author) from each post file
- **Dynamic routing** — one TanStack route serves every post by slug

## Markdown features

Regular Markdown works as expected. You can use **bold**, _italic_, and `inline code`.

> Blockquotes are styled too. Use them for callouts or pull quotes.
